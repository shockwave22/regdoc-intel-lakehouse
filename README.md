# regdoc-intel-lakehouse

An end-to-end Databricks Lakehouse that ingests unstructured regulatory documents
(policies, contracts, and procedures), extracts and normalizes text, enriches it
with metadata and classifications, and powers a governed retrieval-augmented
generation (RAG) assistant for compliance teams.
